# Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat halaman web. HTML mendeskripsikan struktur dari sebuah halaman web melalui elemen-elemen. Elemen-elemen HTML memberi tahu browser bagaimana menampilkan konten.

## Apa itu HTML?

HTML adalah singkatan dari HyperText Markup Language. HTML bukan bahasa pemrograman, melainkan markup language yang digunakan untuk mendefinisikan struktur konten web. HTML terdiri dari serangkaian elemen yang digunakan untuk mengelilingi berbagai bagian konten untuk membuatnya muncul dengan cara tertentu atau bertindak dengan cara tertentu.

>   HTML bukan bahasa pemrograman, tetapi bahasa markup yang menentukan struktur konten web Anda.


## Struktur Dasar HTML

Sebuah dokumen HTML selalu dimulai dengan deklarasi jenis dokumen `<!DOCTYPE html>` dan diakhiri dengan tag `</html>`. Struktur dasar HTML terdiri dari:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Ini adalah Heading</h1>
    <p>Ini adalah paragraf.</p>
  </body>
</html>
```

Mari kita lihat struktur di atas:

- `<!DOCTYPE html>`: Deklarasi tipe dokumen yang membantu browser menampilkan halaman web dengan benar.
- `<html>`: Elemen root dari halaman HTML.
- `<head>`: Berisi informasi meta tentang dokumen.
- `<title>`: Menentukan judul untuk halaman (yang ditampilkan di tab browser).
- `<body>`: Berisi konten yang terlihat dari halaman web.
- `<h1>`: Mendefinisikan heading besar.
- `<p>`: Mendefinisikan paragraf.

## Elemen HTML

Elemen HTML biasanya terdiri dari tag pembuka dan tag penutup, dengan konten di antaranya:

```html
<tagname>Konten...</tagname>
```

### Contoh Elemen HTML Umum

Berikut adalah beberapa elemen HTML yang sering digunakan:

#### Headings

HTML menyediakan elemen heading dari `<h1>` hingga `<h6>`:

```html
<h1>Ini adalah heading 1</h1>
<h2>Ini adalah heading 2</h2>
<h3>Ini adalah heading 3</h3>
<h4>Ini adalah heading 4</h4>
<h5>Ini adalah heading 5</h5>
<h6>Ini adalah heading 6</h6>
```

#### Paragraf

Paragraf didefinisikan dengan tag `<p>`:

```html
<p>Ini adalah paragraf.</p>
<p>Ini adalah paragraf lain.</p>
```

#### Link

Link didefinisikan dengan tag `<a>`. Atribut `href` menentukan URL tujuan:

```html
<a href="https://www.example.com">Kunjungi Example.com</a>
```

#### Gambar

Gambar didefinisikan dengan tag `<img>`. Tag ini tidak memiliki tag penutup. Atribut `src` menentukan URL gambar dan atribut `alt` memberikan teks alternatif:

```html
<img src="img_girl.jpg" alt="Foto Perempuan">
```

## Atribut HTML

Atribut menyediakan informasi tambahan tentang elemen HTML. Atribut selalu ditentukan dalam tag awal dan biasanya terdiri dari pasangan nama/nilai seperti: `nama="nilai"`.

### Contoh Atribut Umum

- `href`: Menentukan URL untuk link
- `src`: Menentukan URL untuk gambar
- `width` dan `height`: Menentukan lebar dan tinggi untuk gambar
- `alt`: Menentukan teks alternatif untuk gambar
- `style`: Menentukan gaya inline CSS untuk elemen
- `lang`: Menentukan bahasa konten
- `title`: Menentukan informasi tambahan tentang elemen

## Mengapa Belajar HTML?

HTML adalah dasar dari semua halaman web. Jika Anda ingin membuat website, Anda harus menguasai HTML. Selain itu:

1. HTML adalah bahasa fundamental untuk pengembangan web
2. HTML bekerja dengan CSS untuk styling dan JavaScript untuk interaktivitas
3. HTML adalah standar web yang didukung oleh semua browser
4. HTML mudah dipelajari dan dipahami

## Langkah Selanjutnya

Setelah Anda memahami dasar-dasar HTML, Anda dapat melanjutkan dengan:

1. Mempelajari elemen HTML lanjutan
2. Belajar CSS untuk mendesain halaman web Anda
3. Belajar JavaScript untuk membuat halaman web interaktif

Selamat belajar HTML!
